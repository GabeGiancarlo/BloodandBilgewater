# Palette

Anchor colors for Blood and Bilgewater. Extend sparingly; keep gameplay readability first.

| Name | Hex | Usage |
|------|-----|--------|
| Deep maroon | `#580000` | Blood, danger, accent shadows |
| Tarnished gold | `#C79E4B` | Loot highlights, UI accents, worn metal |
| Moonlit teal | `#3E5B6F` | Night water, fog, cool ambient fill |

Supporting neutrals (dark stone, weathered wood, sea foam) should harmonize with these anchors rather than competing with them.
