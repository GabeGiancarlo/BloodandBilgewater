# Blood and Bilgewater — Unity Project

Dark, gothic, nautical, SNES-inspired pirate action-RPG. This is the **2D Unity** project
(migrated from the original Rust/Bevy prototype, now archived in `../Archive/BevyReference/`).

## Purpose

A 2D top-down pirate action-RPG of cursed seas, crew roles, ruins, ocean tiles, and a haunted
archipelago. Art is authored in **Aseprite** and imported directly by Unity.

## What to commit

- `Assets/`, `Packages/`, `ProjectSettings/`
- All `.meta` files (they hold stable asset GUIDs)
- `.aseprite` source art (tracked via Git LFS)

## What NOT to commit

- `Library/`, `Temp/`, `Obj/`, `Logs/`, `UserSettings/` (generated — regenerated by Unity)
- Generated `*.csproj`, `*.sln`/`*.slnx`, `*.user`

These are ignored by the root `../.gitignore`.

## How Aseprite assets flow into Unity

1. Create/edit a `.aseprite` and save it under `Assets/_Project/Art/Aseprite/...`.
2. Unity's **2D Aseprite Importer** auto-generates sprites/animations on focus/refresh.
3. No manual PNG export is required. See `../docs/ASEPRITE_UNITY_WORKFLOW.md`.

> The 2D Aseprite Importer is **not yet installed** — add it via Package Manager. See the
> workflow doc for the package list.
